Exemple d'utilisation


local MyClass = Class:registerClass("MyClass", {
  init = function(self)
    self.__private.value = 42
  end
})
local obj = MyClass()
print(obj:DebugInfos())

API

  • includeHelper : Fusionne des tables de classes.
  • debugInfo : Affiche les infos de debug d'un objet.
  • accessor : Génère des getters/setters.
  • include : Inclut une classe dans une autre.
  • clone : Clone un objet.
  • super : Appelle une méthode parente.
  • overloadOperators : Surcharge les opérateurs.
  • new : Instancie une classe.
  • registerClass : Enregistre une nouvelle classe.
